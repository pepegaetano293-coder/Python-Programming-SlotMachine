# Python-Programming-SlotMachine
Slot Machine Simulator (Python).

## Descrizione
Sviluppo di un simulatore di Slot Machine a tre rulli basato sulla gestione di probabilità uniformi e input iterattivo.
Il progetto si focalizza sulla gestione degli stati di gioco e sull'implementazione di una logica di verifica delle condizioni
di vittoria (match di simboli).

## Caratteristiche principali:
**Randomizzazione:** Utilizzo della libreria random per la generazione casuale dei simboli su tre rulli indipendenti.
**Sistema di betting:** Gestione semplificata del bilancio del giocatore (tramite monete) basata sull'esito delle giocate.
**Controllo degli Input**: Gestione dei flussi di gioco tramite interazione testuale con l'utente.

## Tecnologie utilizzate
**Linguaggio:** Python.
**Ambiente:** Visual Studio Code.
